# Yes Bank Stock Price Prediction using Machine Learning

## Project Overview
This project analyzes historical Yes Bank stock price data using Exploratory Data Analysis (EDA) and Machine Learning techniques. The objective is to understand stock price trends, identify relationships among different price variables, and build a predictive model for forecasting the closing stock price.

## Dataset
- Dataset: Yes Bank Stock Prices
- Records: 185 monthly observations
- Features:
  - Date
  - Open
  - High
  - Low
  - Close

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Feature Engineering
6. Model Building
7. Model Evaluation
8. Conclusion

## Machine Learning Models
- Linear Regression
- Random Forest Regressor

## Results
- Strong correlation observed among Open, High, Low, and Close prices.
- Random Forest Regressor achieved high prediction accuracy.
- The developed model can assist in forecasting future closing stock prices.
